# CircleBar

> A circle bar is a type of progress bar that represents progress as a percentage of a full circle. This type of progress bar can be used to provide visual feedback on the progress of a task or process, and is often used in user interfaces for games, fitness apps, and other applications where progress is represented as a circular or radial element.
