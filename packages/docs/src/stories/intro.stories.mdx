# Zero UI React UI library

Zero UI is a modular and extensible React components library powered by Radix UI and Stitches.

## Quick Start

```bash
# Install with npm
npm install @zero-ui/core @zero-ui/helpers @zero-ui/icons

# Install with yarn
yarn add @zero-ui/core @zero-ui/helpers @zero-ui/icons

# Install with pnpm
pnpm add @zero-ui/core @zero-ui/helpers @zero-ui/icons
```
