# 01Group React UI Library

[![npm](https://img.shields.io/npm/dm/@01group-ui/core)](https://www.npmjs.com/package/@01group-ui/core)

A React UI component library from Zero One Group.

Visit the [project page](https://github.com/zero-one-group/zero-ui) page for more detailed information.

> _**Warning: this project currently still WIP and alpha release!**_

## Installation

```bash
# Install with npm
npm install @01group-ui/core

# Install with yarn
yarn add @01group-ui/core

# Install with pnpm
pnpm install @01group-ui/core
```
